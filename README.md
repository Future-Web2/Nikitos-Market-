# Образовательная Демонстрация XSS (Reflected Cross-Site Scripting)

ЭТОТ ПРОЕКТ СОЗДАН ИСКЛЮЧИТЕЛЬНО В ОБРАЗОВАТЕЛЬНЫХ ЦЕЛЯХ.

## Описание
Этот проект представляет собой имитацию онлайн-магазина ("ShopMarket"), который содержит **намеренную уязвимость XSS (Inter-site Scripting)** в функции поиска.

**Цель:** Продемонстрировать, как отсутствие санитарной обработки (sanitization) входных данных может привести к выполнению вредоносного JavaScript-кода в браузере пользователя.

## Уязвимость
Уязвимость находится в строке поиска. Введённый пользователем запрос отображается на странице результатов без экранирования HTML-тегов.

**Как исправить (Mitigation):**
В реальном приложении ВСЕГДА экранируйте вывод пользовательских данных. Например, используя библиотеки для экранирования HTML (escape-html) или встроенные механизмы шаблонизаторов (в EJS по умолчанию `<%= %>` экранирует, но мы намеренно используем `<%- %>` для создания уязвимости).

## Запуск
1. `npm install`
2. `npm start`
3. Откройте `http://localhost:3000`
